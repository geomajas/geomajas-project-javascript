Basic map and layer events

When the Geomajas map is initialized, the map is build in a number of steps. E.g. a request is sent to the server to ask for the map's configuration. The layers are added one by one to the map. Each time a LayersModelChangedEvent is fired. When the map has been completely initialized, a MapInitializationEvent will be fired.
